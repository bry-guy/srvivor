# Discord Bot MVP Delivery Plan

Status: `done`

## Goal

Ship the initial standalone Discord bot that reads Castaway draft state from `castaway-web`.

## Delivered scope

- slash commands for scores, leaderboards, drafts, and instance context
- local state for saved user and guild defaults
- Castaway API client integration
- local development workflow through the monorepo stack
